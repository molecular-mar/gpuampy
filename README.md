## Installation üõ†Ô∏è

The package will be included into a Python repository in the near future. Meanwhile, follow the next steps:

1. Clone this repo:
   ```sh
   git clone https://github.com/molecular-mar/gpuampy.git
   ```
2. (For users) Enter the folder and install with pip (using a local env is **highly** recommended):
   ```sh
   cd gpuampy
   pip install .
   ```
2. (For developers) Install using the editable/developer option:
   ```sh
   pip install -e .
   ```
